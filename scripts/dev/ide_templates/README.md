# IDE Templates

These templates provide baseline settings and extension/plugin recommendations for common IDEs used with this repository.

Available templates:
- `vscode/`: VS Code settings and extension recommendations.
- `cursor/`: Cursor (VS Code-compatible) settings and extension recommendations.
- `windsurf/`: Windsurf (VS Code-compatible) settings and extension recommendations.
- `zed/`: Zed settings and extension recommendations.
- `jetbrains/`: JetBrains IDE settings checklist and plugin recommendations.

Usage:
1. Pick the folder for your IDE.
2. Copy template files into your local IDE config.
3. Adjust personal preferences without committing machine-specific paths.
