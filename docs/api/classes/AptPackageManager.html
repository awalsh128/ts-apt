    Class AptPackageManager

    APT-backed package manager implementation.

    IMPORTANT: All operations are serialized to avoid concurrent access to APT lock files. This is achieved via a global mutex and flock-based locking for real command execution.

    ProtectedrunAptCommand

    • runAptCommand(
          binary: BinaryMeta,
          baseArgs: string[],
          timeoutMs: number,
      ): Promise<CommandResult>

      Executes an apt command with fixed non-interactive behavior.

      NOTE: Also ensures that only one apt command is executed at a time across all processes via a global lock, otherwise apt will fail with a lock acquisition error.

      Effective invocation: binary.path ...binary.defaultArgs ...baseArgs

      Example composed command:

      /usr/bin/apt-get --quiet=0 --force-yes install --fix-broken curl

      Parameters

      • binary: BinaryMeta

        Binary metadata (path plus default args).

      • baseArgs: string[]

        Command arguments before fixed flag expansion.

      • timeoutMs: number

        Command timeout in milliseconds.

      Returns Promise<CommandResult>

      Command result with captured process output.
